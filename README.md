# Sophium - Especificação de Requisitos de Software

## Sobre o Projeto

O Sophium é uma proposta de aplicativo mobile voltado para organização de estudos, utilizando conceitos de gamificação para incentivar a disciplina, a constância e o engajamento dos estudantes.

Este projeto foi desenvolvido por mim e meu grupo como atividade da disciplina de Engenharia de Requisitos de Software da PUC Minas, e teve como objetivo: aplicar técnicas de levantamento, análise e especificação de requisitos para um sistema de software.

O foco do trabalho não foi a implementação da aplicação, foi a definição detalhada de como o sistema deveria funcionar, servindo como base para um possível desenvolvimento futuro.

---

## Problema

Muitos estudantes enfrentam dificuldades para manter uma rotina de estudos consistente devido à procrastinação, falta de motivação e dificuldade em acompanhar sua evolução ao longo do tempo.

O Sophium foi idealizado para ajudar na resolução desses problemas por meio de mecanismos de gamificação, sistemas de recompensas e acompanhamento de desempenho.

---

## Proposta da Solução

O sistema permite que os usuários:

* Criem e gerenciem metas de estudo;
* Participem de comunidades de aprendizagem;
* Acumulem pontos de experiência (XP);
* Evoluam de nível conforme seu progresso;
* Recebam recompensas virtuais;
* Acompanhem rankings entre membros das comunidades;
* Monitorem sua evolução de forma contínua.

---

## Principais Requisitos Funcionais

* Cadastro e autenticação de usuários;
* Recuperação de senha por e-mail;
* Visualização e gerenciamento de metas;
* Participação em comunidades de estudo;
* Sistema de ranking baseado em XP;
* Progressão automática de níveis;
* Distribuição de recompensas virtuais;
* Edição de perfil do usuário.
